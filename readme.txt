Key inputs

m - change method
n - increase n
b - decrease n
l - increase timestep
k - decrease timestep
p - increase theta
o - decrease theta
q - decrease number of runs before output
w - increase number of runs before ouptut
r - restart sim and apply changes

All variables can easily be changed in the setup.txt file.

Output will be output to the output folder.
